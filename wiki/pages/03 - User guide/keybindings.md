{{#vardefine:chapter\|B}} {{#vardefine:figure\|0}} This appendix lists keybindings (also known as *Keyboard shortcuts* or *hotkeys*) that can be used in the Gramps dialogs as an alternative to using the mouse.

Some keybindings require the user to press a single key or a sequence of keys one after the other. Other keybindings require pressing and holding several keys simultaneously.

![[_media/windows_180x160.png]] ![[_media/macos_200x200.png]] ![[_media/Linux_220x261.png]] ![[_media/Bsd_daemon.png]]}}

## Access Key Shortcuts

In general with most programs you can use the access key shortcuts for accessibility to navigate or access menus. The most useful of these allow you navigate within an interface that has a lot of entry boxes and separate sections like the Person Editor window. The access key shortcuts are shown as underlined letters or numbers when you press and hold the key and then the corresponding letter or number that is shown underlined this will move the focus of the interface to that field or if it is a menu command it will be run.

For example on the Gramps Person Editor window to go the bottom half of the editor that shows the various tabs you can use the keyboard to press the key and then press the letter “” twice to go to the “” tab.

## Context menu

A [context pop-up menu](https://wikipedia.org/wiki/Context_menu) shows limited number of choice related to the currently selected item in a Graphical User Interface. The menu appears with right-click on an item in operating systems with a 2 or 3 button mouse. In the one button mouse macOS, a -click will bring up the [Contextual Menus](https://developer.apple.com/design/human-interface-guidelines/macos/menus/contextual-menus/).

Keybindings:

- In Windows, the **[Menu Key](https://wikipedia.org/w/index.php?title=Menu_key)** (aka **application key**) will show the context menu for the active field. Alternately, the + [keyboard shortcut](https://wikipedia.org/wiki/Keyboard_shortcut), or sometimes ++
- In macOS, users will need to enable the Universal Access settings in the System Preferences.

See [Context Menu](wiki:Gramps_{{man_version}}_Wiki_Manual_-_Settings#Context_menu)
