**Investigue, organize e partilhe a sua árvore genealógica com o Gramps.**

O Gramps é um projecto comunitário livre e gratuito. Trabalhamos para produzir um programa de genealogia que seja tão intuitivo para genealogistas amadores, como completo para profissionais. É um projecto criado, desenvolvido e gerido por genealogistas.

{{-}}

## Sinopse

Este **manual [wiki](wiki:Gramps_Glossary#wiki) do Gramps** para a versão é o manual em linha do utilizador, descrevendo a maioria das funções do Gramps, uma base de dados para recolha de informação genealógica. A partir de --, a [última versão do Gramps](wiki:Download/pt) é a . Tanto o Gramps como o seu manual wiki [foram traduzidos em múltiplos idiomas](wiki:User_manual_translations).

Também pode transferir o **manual completo da versão 5.1** ([Inglês, 20 MiB PDF](wiki::File:Gramps5.1UserManual.pdf) e [outros idiomas](wiki:User_manual_translations#Gramps_6.0.x_Manual)), actualizado a 27-08-2023. Para obter versões anteriores, veja as [versões arquivadas](wiki:User_manual_translations#superseded), feitas para [versões anteriores do Gramps](wiki:Previous_releases_of_Gramps) (procure na página por "*download*" para localizar versões para transferir).

O **manual de extensões 5.2 em PDF** está disponível para revisão [aqui (Inglês 18 MiB)](wiki::File:GrampsAddonsUserManual5.2.pdf), última actualização a 10-6-2024.

Para explorar outras fontes sobre o Gramps, veja o [portal de informação](wiki::Portal:Using_Gramps).
